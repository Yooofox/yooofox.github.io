npm修改国内镜像源
发表于2021-11-16
npm修改国内镜像源npm 是随同 Node.js 一起安装的包管理工具，能解决 Node.js 代码部署上的很多问题。但在国内直接使用 npm 的官方源是十分缓慢的，因此就需要将官方源替换为国内的镜像源，对访问速度会有很大的提升。 以下是修改 npm 源的命令1npm config set registry [address] 其中，[address]为镜像源的地址 例如 淘宝镜像1npm config set registry https://registry.npm.taobao.org 华为镜像1npm config set registry https://mirrors.huaweicloud.com/repository/npm/ 查看是否修改成功1npm config get registry 如果修改的是淘宝镜像源，那么应该返回https://registry.npm.taobao.org，表示修改成功。
